# GitSample

Git の練習用リポジトリです。

## 目的

このリポジトリは、Git や GitHub の基本操作（ブランチ作成、コミット、プルリクエストなど）を練習するためのサンプルプロジェクトです。

## 使い方

```bash
# リポジトリをクローン
git clone https://github.com/ogawatakahiro930-rgb/GitSample.git

# ブランチを作成して作業
git checkout -b feature/your-feature

# 変更をコミット
git add .
git commit -m "変更内容のメッセージ"

# プッシュしてPRを作成
git push origin feature/your-feature
```
